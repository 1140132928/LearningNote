> **问题：推荐信该怎么准备？（两封左右）**
>
> 答案：（1）应该找谁写？
>
> ​    推荐信最好找一个了解你的（了解你的老师很重要，这样才有可写的内容，要不然老师只能写套话了）、有分量的（教授呀，副教授呀之类的大人物们总是没有错的）推荐人。　
>
> （2）推荐信里写哪些内容呢？
>
> 　　A.推荐学生的具体能力，尤其是专业能力，学术科研的能力或者是潜力，附带推荐一下学生为人处世的能力（例如：工作经验，学习精神，组织能力，品行及个性方面）。注意说辞应与其他申请材料如成绩单、获奖证书等相呼应，不要出现矛盾而使人误会。
>
> 　　B.为了增加推荐信的真实性，保证客观公正，可以写少量并不重要但很中肯的小缺点，这不失为一种聪明的策略；人无完人，教授们要的不是完人，而只是学生，一个普通但足够优秀的学生。
>
> 　　C.推荐信的语言风格应该清晰流畅、言简意赅
>
> 　　D.推荐信切忌流于形式，内容空洞 (ps：那种百度出来一堆的“×××同学品学兼优，乐于助人，善良开朗……”这种烂大街的推荐信最好就不要用了，没有诚意！！！一看就是网上扒下来的！！)
>
> 　　E.交待与被推荐者的关系，体现该推荐人和申请者是如何相识的，是什么关系，并确实了解申请者的情况，这样才合乎情理和逻辑。

## 个人陈述

  [知乎参考](https://www.zhihu.com/question/30606095)

> **问题：个人陈述该怎么写？**
>
> 答案：个人陈述是保研时目标院校的老师了解申请者的主要途径，好的个人陈述能够让申请者脱颖而出。从申请表个人简历等中老师更多了解的是成绩、论文、获奖经历等，而老师通过个人陈述可以更为详细的了解申请者的申请动机、科研潜能、思维方式等。个人陈述中应该包括个人学术背景（学校专业以及专业课学了什么等）、科研经历（参加了哪些科创项目，成果有哪些等）、社会工作经历（担任过哪些职位，任职期间做了哪些事，遇到问题怎么去解决等）、读研的计划等。但个人陈述不论结构、形式如何变化，其核心要素始终是固定的，即说清楚两个问题“我为什么要选择”和“我为什么能选择”。
>
>    “我为什么要选择”着重说明的是自己选择继续读研的原因，以及为什么要选择这个专业和学校。想清楚自己为什么要读研。只有自己坚定了读研的目标明确了读研的意义，才能够说服别人相信自己是一块读研的材料。其次，需要说明自己为什么选择这个专业。选择的理由可以是出于自身的性格特点，也可以是自身对某一领域某些问题的兴趣。总之要让老师明白自己对专业的选择不是因为一时头脑发热，而是经过深思熟虑的结果。最后要说明自己为何会选择这所学校，包括学科实力、未来发展前景甚至饮食习惯、气候等因素。
>
>    “我为什么能选择”则类似于人力资源管理中“胜任力”的概念，意在证明自己有相应的学术能力能够胜任研究生这一角色。即自己大学期间的科研经历，以及为了以后能够从事科研活动所做过的努力。
>
> **注：**个人陈述一般都有模板，而且对字数有一定要求，写之前应该注意一下相关要求。

### 核心点

突出个人特点

+ 学术背景（10%）
  + 开头致谢
  + 基本信息：本科院校，学分绩点，院系排名，课程学分总数。罗列几门分数很高的课程，顺序随意。最后一句深刻感悟。
  + 有联系好的导师，也可以写上
+ 科研实力综述（70%）：
  + **重要原则**（以下三种方法都适用）
    + 事件表述使用=扮演角色+内容概述+重要结论+获得成就
    + 用事件和数据表示能力，而不是单纯说自己有（怎么说服别人）
    + 与学科能力无限靠近，取官网查询学科能力
    + **体现独立思考**：牛人很多，必须要有亮点（这点干啥都能用）
  + 时间轴法：体现清晰规划，大一到大三
  + 能力分类法：针对学科能力，挑出【掌握】【熟悉】要求的能力，分类阐述
    + 科研能力：**发现问题、查阅文献、总结归纳、实验设计、组织实施、科研表达**
  + 自问自答法：从导师的角度思考问题
    + 是否具备科研能力？：发现，提出，解决，查询等等（美赛过程）
    + 是否具备某种学科能力（必须和专业相关）？（突出某种能力）
    + 为啥报这个学科和学院？
+ 研究生规划（20%）
  + 若有明确读博意向，写上加分
  + 时间轴法
  + 分类法
  + 最后再次致谢
+ check list（多检查，可让学长学姐看看修改）
  + **错别字**
  + **字数不超限制**
  + 学校logo
